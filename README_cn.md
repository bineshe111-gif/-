项目简介：
本作品聚焦无人化无线充电停车场的安全保障需求，搭建了子母车协同作业、多模块联动的金属异物检测与清除系统，系统分为无线充电区域监测模块、RDKs100 母车导航模块、RDKx5 子车清除模块及毫米波雷达监测模块。无线充电区域监测模块基于 STM32F407 控制对称式线圈结构，实时感应区域内金属异物并向子母车发送触发信号；RDKs100 母车接收信号后自主导航至指定充电区域，搭载 RDKx5 子车完成转运部署。RDKx5 子车搭载 YOLO 模型对充电板异物进行视觉定位与图像切割，将图像数据回传至母车；母车通过大模型分析图像特征，检测异物是否为金属并将结果反馈给小车，子车依据结果对金属异物执行吸附清除操作。毫米波雷达监测模块实时识别靠近无线充电区域的物体属性，若检测到生物靠近，立即触发无线充电区域断电，避免磁场对生物体造成危害；若检测到生物体远离，则重新开启充电模式。

项目系统介绍：
1. 无线充电装置：
考虑到实际汽车底盘高度约为10-25cm之间，团队基于STM32f407开发板搭建了模拟的无线充电装置高度为17cm，长宽分别为35、25cm。用于充电的发射和接收线圈分别粘贴在两块亚克力板之间。接收线圈连接了用于显示充电状态的LED，正常充电下LED点亮。
2. 金属异物、生物体检测装置：
系统在硬件上利用反向串联式对称线圈检测位于充电区域内的金属异物的初步检测，利用毫米波雷达来检测接近充电区域的活体异物。汽车进行无线充电时，两板间的交变磁场会使金属物品产生涡流效应，金属异物发热并产生一个反向磁场；又因为充电区域中出现的金属异物一般体积微小，不会出现大型金属物品导致充电面板被全部覆盖的现象，因此我们采用反向串联式对称式线圈来检测：利用线圈感应金属物品发出的磁场以产生电压，再将对称线圈的电压进行差分、整流、放大，当对称线圈的电压差超过阈值时，判定为存在金属异物，随后立即停止充电，同时通过Lora通信模块发送信号给清除系统，派出子母式异物清除机器人清理异物。清除完闭后，清除机器人向系统发出清除完毕指令，随后系统重新恢复充电。
同时，我们还在无线充电装置周边安装了HLR12毫米波雷达，检测周边移动物体距离雷达的位置以及运动的速度。通常来说，生物体的移动速度变化较为连续，而非生物体的速度存在突变现象。综合考虑位置距离与物体当前的速度来判断该移动物体是否为生物体，是否会对充电造成影响，若判断对充电有影响，系统立即发出驱赶警报，并降低充电功率，随时准备停止充电，当活体干扰消失后，系统等待一段时间，若无新的报警信号，则重新恢复原来的充电速率。
3. 子母异物清除机器人：
团队基于Originbot和RDKs100自主设计了一款子母式异物清除机器人，母车使用RDKs100和stm32开发板，子车使用RDKx5开发板，如图6所示。
在形态选择方面，子车采用差速驱动车型，为两轮结构。两轮平衡车结构更有利于子车在低矮受限的空间下作业。子车通过两轮差速控制可实现原地旋转，在狭窄空间内灵活调头；子车利用倒立摆原理主动调节重心，适应停车场不平路面或突发障碍，减少卡顿风险。两轮结构可有效降低子车的直立高度，同时简化了其机械结构。在排异结构选型方面，综合考虑成本与清除效率，子车配备了电磁吸附装置以吸附铁磁性金属异物。当子车通过视觉识别到异物后，继电器控制电磁铁通电，子车利用磁铁吸附异物，将其带出充电区域。
母车选用四轮底盘构型，整车重心偏低、行驶稳定性优异，载重承载性能优良，可满足大负载作业需求；设备能够在大型无线充电停车场内实现全域自主巡逻与导航，适配场内各类复杂路面，有效提升整车通行能力。车体专为子车收纳与进出设计电机驱动式升降挡板：子车执行异物清理任务时，挡板下降形成斜坡通道供子车驶出；作业完成后，子车沿斜坡返回母车舱体，挡板自动抬升并锁止至固定位置。异物清除系统部署至全新作业场地时，母机器人将先开展全域环境建模工作。母车搭载激光雷达，依托 SLAM 算法完成场地高精度全局地图构建，完整标注所有车位坐标；随后在地图中标定基站与各无线充电区域的坐标边界，最终将完整地图数据存储至车载处理单元备用。
子母机器人金属异物的识别和清除：当新能源汽车驶入充电区域并驻车后，无线充电系统通过电磁感应装置扫描充电区域，若检测到异物信号，则触发“检测”信号并通过Lora通信模块将信号发送给母车，并停止无线充电。母车接收到信号后，以恒定速度（如5cm/s）向指定停车位中心移动，到达指定充电点位后，母车停止运动并放下挡板，子机器人驶入充电区域完成现场图像采集，通过YOLO模型裁切图像中的异物区域并回传至母车，母车利用大模型判断后指引子车进行清除。清除完毕后，子车返回母车，母车返回基站。
